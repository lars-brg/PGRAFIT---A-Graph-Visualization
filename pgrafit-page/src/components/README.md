# ⚛️ Componentes (`src/components`)

Este diretório armazena todos os componentes React da aplicação. A organização é crucial para a reusabilidade e a manutenção do projeto.

## Estrutura Interna

1.  **`app/`**: Componentes específicos da Aplicação.
2.  **`ui/`**: Componentes de Interface de Usuário (UI) genéricos.
